## 安装

### docker安装

拉取镜像

```shell
docker pull whyour/qinglong:latest                                             
```

运行容器，`  -v /home/ql/data:/ql/data`是将数据挂载到宿主机 /home/ql/data 目录

```
docker run -dit \
  -v /home/ql/data:/ql/data \
  -p 5700:5700 \
  --name qinglong \
  --hostname qinglong \
  --restart unless-stopped \
  whyour/qinglong:latest
```

### 登录

http://192.168.10.2:5700
