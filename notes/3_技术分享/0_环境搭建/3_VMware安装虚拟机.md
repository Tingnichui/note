#### 安装完成

1. 登录系统 账号root 密码自己设置的

2. 检查是否联网 

   ```bash
   ping www.baidu.com
   ```

3. 关机做快照，方便后面虚拟机回退或者克隆，一定要用命令关机，不要用VMware的！！！

   ```
   shutdown -h now
   ```
